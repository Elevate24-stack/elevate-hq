How Dialpad Integration Works
1. API Key — Enter your Dialpad API key above
2. Webhooks — Dialpad sends call/SMS events to your webhook URL
3. Auto-logging — Every call, transcript & SMS appears here automatically
4. AI Analysis — Claude reads each transcript and extracts action items
5. Task creation — Follow-ups auto-added to your Task Inbox
