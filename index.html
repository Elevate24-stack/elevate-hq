After entering your credentials above, click Authorize to open Google's permission screen. Sign in with [email protected] and allow access. You'll be redirected back — copy the authorization code and paste it below.

🔗 Website Contact Form → Lead Pipeline

When someone fills out your contact form on GaElevate.com, leads flow here automatically

How this works: Your GaElevate.com contact form sends data to a webhook URL. This app receives it and automatically creates a new CRM lead. No manual entry needed.
https://elevate24-stack.github.io/elevate-hq/?lead=1&name={name}&email={email}&phone={phone}&message={message}&source=website
Replace {name}, {email}, {phone}, {message} with your form field variables
📋 Setup Instructions
GoDaddy Website Builder: Go to Forms → Contact Form → Settings → Notifications → Add Webhook URL above
WordPress (Contact Form 7): Install "CF7 to Webhook" plugin → paste URL
Squarespace: Form Block → Storage → Webhooks → paste URL
Any HTML form: Add action="URL" to your form tag with method GET

📧 Email Integration Setup

Connect Gmail and Outlook to pull leads automatically into your pipeline

Full email automation requires a Node.js backend server. Below is your complete setup guide and what each part costs.
📧 Gmail
Google Workspace API
1. Go to console.cloud.google.com
2. Create project → Enable Gmail API
3. Create OAuth credentials
4. Set up Pub/Sub webhook → points to your Node.js server
5. Server catches email → creates CRM lead automatically
FREE — Google API is free for this usage
📨 Outlook
Microsoft Graph API
1. Go to portal.azure.com
2. App registrations → New registration
3. Add Mail.Read permission
4. Set up Graph API subscription webhook
5. Server catches email → creates CRM lead automatically
FREE — Microsoft Graph API is free
🖥️ Backend Server Options (to run webhooks)
Railway.app — Deploy Node.js free tier, ~$5/mo after. Easiest setup. ⭐ Recommended
Render.com — Free tier available, spins down after inactivity
Vercel — Great for serverless functions, free tier
Supabase — Postgres database + edge functions, free tier
📞 AI Receptionist (Dialpad)
Vapi.ai — Best ultra-low latency voice AI. $0.05/min. Configure Dialpad to forward to Vapi SIP endpoint.
Bland AI — Specialized for business calls, good for scheduling. ~$0.09/min
Retell AI — Great customization, $0.07/min
All three connect to your Node.js brain via webhooks — caller asks a question, Vapi calls your server, server pulls from your database, Vapi speaks the answer back. Sub-1 second latency.
📞 Vapi AI Receptionist — Setup
Step 1 — Go to vapi.ai → Create account → Dashboard
Step 2 — Create Assistant → Name: "Elevate Receptionist"
Step 3 — Set System Prompt (click Generate below)
Step 4 — Set Server URL to your Railway webhook:
https://elevate-hq-production.up.railway.app/webhook/vapi
Step 5 — In Dialpad: Settings → Integrations → Forward calls to Vapi SIP endpoint
Step 6 — Enter your Vapi API key below and activate
📧 Gmail Webhook — Activate
1. Go to console.cloud.google.com → New Project: "ElevateHQ"
2. Enable Gmail API + Cloud Pub/Sub API
3. Create Pub/Sub Topic → Add subscription → Push endpoint:
https://elevate-hq-production.up.railway.app/webhook/gmail
4. Create OAuth credentials → Authorize Gmail → Watch inbox
📨 Outlook Webhook — Activate
1. Go to portal.azure.com → App Registrations → New
2. Add permission: Microsoft Graph → Mail.Read
3. Create Graph subscription → notification URL:
https://elevate-hq-production.up.railway.app/webhook/outlook
4. Subscribe to: /me/messages — change type: created
